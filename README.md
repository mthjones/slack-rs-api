# slack-rs-api

[Slack][slack] Web API interface.

[![Build Status][ci-img]][ci-url] [![Crates.io][crates-img]][crates-url] [![License][license-img]][license-url]

[Documentation][docs]

# Usage

Add this to your `Cargo.toml`:
```toml
[dependencies]
slack_api = "0.18.0"
```

and this to your crate root:

```rust
extern crate slack_api;
```
